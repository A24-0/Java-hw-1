```Занятие 1. 31.01.2024
Задание 0
    Установите Java JDK версии не ниже 8. Установите IDE для написания
    и сборки кода, например IntelliJ IDEA.
Задание 1
    Високосным является год, номер которого делится на 4, но не делится
    на 100, например 2004, но не 1900, либо номер года делится на 400, например 2000. Напишите программу, которая считывает из стандартного
    потока целое положительное четырёхзначное число и выводит строку
    Leap, если год високосный, либо No Leap, если год не високосный.
Задание 2
    Напишите программу, которая получает из стандартного потока целое
    положительное число, не превышающее 2^31, и меняет в нём старшие два
    байта и младшие два байта местами с помощью битовых операций. Результат выведите в стандартный поток.
Задание 3
    Рассмотрим последовательность натуральных чисел, устроенную следующим образом: a(n+1) = (a(n))/2, если a(n) - чётно, a(n+1) = 3 × a(n + 1), если a(n) -
    нечётно. Такая последовательность содержит один очевидный повторяющийся набор значений: 1 → 4 → 2 → 1. Для огромного количества чисел
    проверено, что последовательность в конечном итоге обязательно приходит к единице. Напишите программу, которая считает из стандартного
    потока ввода натуральное число, не превышающее 2^31 и будет применять описанную процедуру к числу до тех пор, пока значение не станет
    равным единице, после чего выведет количество итераций, которое потребовалось для достижения заданного значения. Например для числа
    15 это значение равно 17. 15 → 46 → 23 → 70 → 35 → 106 → 53 → 160 →
    80 → 40 → 20 → 10 → 5 → 16 → 8 → 4 → 2 → 1
Задание 4
    Пусть задан массив из N (0 < N < 10000) целых чисел, каждое из которых равно либо 0, либо 1. Напишите программу, которая подсчитает
    количество единиц и количество нулей в этом массиве.
Задание 5
    Пусть задан массив из N (0 < N < 2^31) чисел от 1 до N. Каждое число
    встречается в массиве ровно один раз, но числа расставлены в случайном
    порядке. Напишите программу, которая за O(N) действий и O(1) дополнительной памяти переставит числа в массиве в порядке возрастания.
Указание
    Ввод в стандартном потоке не нужно проверять на ошибки.
